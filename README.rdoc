= Apply.io

=== Objective
A web app that allows users to view applications for schools they have applied for.

=== Installation

Fork the repo

And then execute:

    $ bundle

=== Server

From your terminal enter

    $ rails server
or

    $ rails s


=== Dependencies

* Rails
* Boostrap
* Faker
* Backbone.js
* Underscore.js
* Gem Faker
* Coffeescript
* Gem Responders
* Gem BCrypt
* Gem Rails-Backbone
